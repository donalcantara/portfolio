TECHNOLOGIES

  • HTML5/CSS3

    I started with the basics like HTML a few years back. In the past year, I've rediscovered the art of HTML and it's importance in website design. It's the backbone of every webpage and I continue to practice it's versatility and structure daily. CSS makes all my websites jump out at everyone who visits! I love playing around with all that CSS has to offer to give my websites that touch that will make you feel like revisiting my websites.

  • Python

    Python is a back-end language that I enjoy immensely. It allows the ease in working with databases to store pertinent information for every website I create. Along with it's helpfulness in communication with databases, it's also taught me the basics in how to create clean code just in case I'm not the only person to touch the code.

  • JavaScript

    Algorithms are what rules today's tech world and it's an interest of mine to create algorithms that not only cater to what website owners desire but make my websites faster and much smarter.

  • MongoDB

    Definitely my favorite database. Speed is a huge factor. I try to use it on all the sites I create!

  • Mongoose

    Mongoose is a huge reason why I feel like MongoDB is my favorite database. Actually, come to think of it, without Mongoose, MongoDB wouldn't be my favorite database, lol.

  • Node.js

    Node Package Modules make life easy. They're great! Node is still new to not only me but to everyone on the internet. It's a great server with lots of options and technology is still being created for Node.js everyday! It's tough to keep up but definitely fun once you find the right NPM you're looking for to help you out. One of my favorites? Nodemon? Oh yeah, Express is pretty awesome, too.

  • Angularjs

    Single page applications are the best, in my opinion. They're clean, efficient and cool. Angular is definitely the way I go when designing the front-end for dynamic web pages. I guess you could say, it's just my style.
